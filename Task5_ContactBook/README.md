# Task 5 - Contact Book

## Description

This project is a command-line based Contact Book Application developed using Python. It allows users to store, manage, search, update, and delete contact information efficiently. The application uses file handling to save contact details permanently.

## Features

* Add new contacts
* View all contacts
* Search contacts by name
* Update contact details
* Delete contacts
* Save contacts to a file
* Load saved contacts automatically when the application starts
* User-friendly menu-driven interface

## Files

* contact_book.py - Main source code
* Contacts.txt - Stores contact information

## Sample Menu

1. Add Contact
2. View Contact List
3. Search Contact
4. Update Contact
5. Delete Contact
6. Exit

## Learning Outcomes

* File handling in Python
* List manipulation
* Searching and updating records
* Function-based programming
* Data storage and retrieval
* Menu-driven application development
