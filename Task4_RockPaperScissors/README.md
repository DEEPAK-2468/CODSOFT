# Task 4 - Rock Paper Scissors Game

## Description

This project is a command-line based Rock Paper Scissors game developed using Python. The user competes against the computer, which randomly selects Rock, Paper, or Scissors. The game determines the winner based on standard game rules and keeps track of scores across multiple rounds.

## Features

* User-friendly menu
* Random computer choice
* Winner determination logic
* Score tracking
* Multiple rounds of gameplay
* Input validation

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock

## Learning Outcomes

* Random number generation
* Conditional statements
* Loops in Python
* User input handling
* Score tracking logic
